---------------------------------------------------------
OVERVIEW
---------------------------------------------------------

Version 0.6 marks the completion of the core CRUD system for the Inventory Management System.

This update introduces complete management of Products, Customers, Purchases and Sales along with automatic stock management, transaction handling, reusable helper functions, improved terminal UI, detailed validations and review screens before committing data to the database.

The application can now manage the complete inventory workflow from purchasing products from suppliers to selling products to customers while automatically maintaining inventory levels.

---------------------------------------------------------

→ Execute SQL
→ Commit

On Failure
→ Rollback
→ Display Error

Finally
→ Close Cursor
