# 📦 Build Log - v0.1
**Release:** v0.1.0 - Project Foundation & First-Time Setup

**Status:** ✅ Completed

---

## 🎯 Overview

Version 0.1 establishes the core architecture of the Smart Inventory Management System. This release focuses on creating a modular project structure, designing the database, implementing a professional first-time setup experience, and preparing the application for executable distribution.

---

## ✨ Features Added

### 📁 Project Foundation
- Created a clean and modular project structure.
- Organized project into dedicated folders for SQL scripts, documentation, screenshots, and modules.
- Added project configuration files including `.gitignore`, `.env.example`, and `README.md`.

### 🗄️ Database Design
- Designed the complete Entity Relationship (ER) Diagram.
- Created relationships between:
  - Users
  - Customers
  - Products
  - Categories
  - Suppliers
  - Sales
  - Sale Items
  - Purchases
  - Purchase Items

### ⚙️ First-Time Setup Wizard
Implemented an interactive setup wizard that runs automatically on first launch.

Features include:
- Detects whether a configuration file already exists.
- Requests MySQL credentials from the user.
- Provides sensible default values.
- Hides password input.
- Validates user input before proceeding.

### 🔌 MySQL Connectivity
Implemented two connection methods:
- Connect using credentials entered during setup.
- Connect using saved configuration for future launches.

### 💾 Configuration Management
- Automatically creates and saves a `.env` configuration file.
- Designed to work in both development mode and packaged executable mode.
- Stores configuration beside the executable when packaged.

### ❌ Error Handling
Added handling for:
- Invalid port numbers.
- Empty passwords.
- MySQL connection failures.
- Configuration save failures.
- Retry or Exit options during setup.

### 🖥️ Terminal Experience
- Added coloured console output.
- Startup banner.
- Success and error messages.
- Improved application flow with loading animations.
