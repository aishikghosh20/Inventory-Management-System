# Smart Inventory Management System
## Build Log - v0.2
Release Date: July 2026

### New Features
- Added First-Time Setup Wizard.
- Automatic detection of existing `.env` configuration.
- Interactive MySQL server configuration.
- Support for custom MySQL Host.
- Support for custom MySQL Port.
- Support for custom MySQL Username.
- Secure password entry with hidden (`********`) mode.
- Optional visible password mode.
- Configuration is automatically saved after a successful connection.
- Configuration file is created beside the executable for portability.
- Automatic loading of saved configuration on future launches.

### Improvements
- Cleaner startup sequence.
- Improved terminal UI with colored status messages.
- Added loading messages during connection.
- Better validation for user inputs.
- Added retry option when MySQL connection fails.
- Added configuration save verification.
- Improved setup workflow for first-time users.

### Bug Fixes
- Fixed `.env` path detection.
- Fixed configuration save issues.
- Fixed password validation logic.
- Fixed password input callable bug (`'str' object is not callable`).
- Fixed multiple setup loop issues.
- Improved exception handling throughout the setup process.

### Internal Changes
- Separated setup logic into `setup.py`.
- Added dedicated configuration save function.
- Added environment configuration loader.
- Added reusable MySQL connection functions.
- Refactored startup flow for better readability.
