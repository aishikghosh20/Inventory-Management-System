# Inventory Management System

## Version 0.4 — Administrator Authentication System

### Release Date
10th July 2026

---

## Overview

Version 0.4 introduces a complete administrator authentication system, allowing secure account creation and login before accessing the Inventory Management System.

This update builds upon the automated database initialization from v0.3 by adding secure user management using bcrypt password hashing and extensive input validation.

---

## New Features

### Administrator Authentication

- Added administrator account creation wizard.
- Added secure administrator login.
- Automatically detects whether an administrator account already exists.
- Automatically launches the administrator setup wizard on first startup.

---

### Password Security

- Passwords are securely hashed using bcrypt.
- Plain-text passwords are never stored.
- Password verification uses bcrypt.checkpw().
- Protection against password disclosure.

---

### User Validation

Implemented validation for:

- Username
  - Cannot be empty
  - Letters, numbers and underscores only
  - Duplicate username detection

- First Name
  - Required field

- Last Name
  - Required field

- Email
  - Email format validation
  - Duplicate email detection

- Phone Number
  - Supports international numbers
  - Length validation
  - Duplicate phone number detection

- Password
  - Minimum 8 characters
  - Password confirmation

---

### SQL Improvements

Added the Users table.

Fields include:

- user_id
- username
- password_hash
- first_name
- last_name
- email
- phone_number
- created_at
- updated_at

---

### Startup Flow

The application now performs the following automatically:

1. Configuration Check
2. MySQL Connection
3. Database Verification
4. Database Creation (if required)
5. Table Verification
6. Table Creation (if required)
7. Administrator Verification
8. Administrator Setup (if required)
9. Secure Login
