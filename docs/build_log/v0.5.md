# Inventory Management System v0.5

## Release Date
11 July 2026

## Overview

Version 0.5 introduces the first complete business module of the Inventory Management System.

This update focuses on Category Management and significantly improves the overall application architecture by introducing role-based navigation, professional terminal interfaces, breadcrumb navigation and a robust relational database design.

The Categories module now serves as the template for future CRUD modules including Suppliers, Products and Customers.

---

## ✨ New Features

### 📂 Complete Categories Module

Implemented the complete CRUD workflow.

Features include:

- Add Category
- View Categories
- Search Category
    - Search by ID
    - Search by Name
- Update Category
    - Name
    - Description
    - Both
- Delete Category

---

### 🔍 Category Validation

Implemented validation for:

- Empty inputs
- Maximum length
- Allowed characters
- Duplicate category names
- Case-insensitive searching
- Case-insensitive duplicate detection

---

### 📋 View Categories

Added sorting support:

- Category ID (Ascending)
- Category ID (Descending)
- Category Name (A-Z)
- Category Name (Z-A)

---

### 🎨 Terminal UI Improvements

Introduced:

- Unicode borders
- Colored terminal output
- Breadcrumb navigation
- Review screens
- Delete confirmation dialogs
- Improved table formatting
- Consistent application styling

---

### 👤 Role-Based Navigation

Added dedicated menus for:

- Administrator
- Manager
- Inventory Staff
- Sales Staff
- Viewer

Role permissions will be implemented progressively in future releases.

---

### 🗄 Database Improvements

Improved the SQL schema by implementing proper foreign key behaviour.

Business entities now use RESTRICT.

Transaction detail tables use CASCADE.

This prevents accidental deletion of important business records while maintaining referential integrity.

---

### ⚠ Error Handling

Improved handling for:

- Invalid user input
- Duplicate entries
- Database exceptions
- Foreign key violations
- Transaction rollback
