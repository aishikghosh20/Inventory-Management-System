-----------------------------------------
OVERVIEW
-----------------------------------------

Version 1.0 marks the first stable release of the Inventory Management System. This version transforms the project from a basic database application into a complete terminal-based inventory management solution with authentication, role-based access control, complete CRUD operations, purchase and sales management, reporting, and administrative settings.

The application is designed using a modular architecture with MySQL as the backend database and Python as the application layer.
