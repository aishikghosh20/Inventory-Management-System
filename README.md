Smart Inventory Management System

Overview

The Smart Inventory Management System is a Python and MySQL-based application designed to help businesses efficiently manage their inventory. The system provides an easy way to organize products, suppliers, customers, purchases, and sales while maintaining accurate stock records.

This project is being developed as part of my software development portfolio to strengthen my understanding of relational database design, SQL, Python programming, and database-driven application development.

---

Features

- User authentication
- Product management (Create, Read, Update, Delete)
- Category management
- Supplier management
- Customer management
- Purchase management
- Sales management
- Inventory tracking
- Low stock alerts
- Product search and filtering
- Sales and inventory reports
